# محلل الأغذية الجزائري - Algerian Food Analyzer

## 🇩🇿 نظرة عامة

تطبيق ويب باللغة العربية مخصص للمستخدمين الجزائريين لتحليل المنتجات الغذائية، مشابه لتطبيق Yuka الفرنسي. يتيح التطبيق للمستخدمين تحميل أو التقاط صور للمنتجات الغذائية لمعرفة مدى صحتها ومحتوياتها.

An Arabic web application for Algerian users to analyze food products, similar to the French Yuka app. The application allows users to upload or take pictures of food products to determine their healthiness and contents.

## ✨ المميزات / Features

### 🔍 تحليل المنتجات
- **تحميل الصور**: إمكانية تحميل صور المنتجات من الجهاز
- **التقاط الصور**: استخدام كاميرا الجهاز لالتقاط صور المنتجات مباشرة
- **السحب والإفلات**: سحب الصور وإفلاتها مباشرة في التطبيق

### 📊 معلومات شاملة
- **التقييم الصحي**: نظام تقييم من 0-100 مع ألوان مختلفة
  - 🟢 ممتاز (80-100): منتج صحي وموصى به
  - 🟡 جيد (60-79): منتج مقبول مع بعض التحفظات
  - 🟠 متوسط (40-59): منتج يحتاج لحذر في الاستهلاك
  - 🔴 ضعيف (0-39): منتج غير صحي، يُنصح بتجنبه

- **معلومات المنتج**: اسم المنتج، العلامة التجارية، والفئة
- **القيم الغذائية**: السعرات الحرارية، الدهون، الكربوهيدرات، البروتين، الملح، والسكر
- **قائمة المكونات**: جميع المكونات الموجودة في المنتج
- **التحذيرات**: تنبيهات حول المواد الضارة أو المثيرة للحساسية

### 🎨 واجهة مستخدم عصرية
- **تصميم متجاوب**: يعمل على جميع الأجهزة (هاتف، تابلت، كمبيوتر)
- **دعم اللغة العربية**: واجهة كاملة باللغة العربية مع دعم RTL
- **ألوان جذابة**: تدرجات لونية حديثة وجذابة
- **رموز تعبيرية**: استخدام Font Awesome للرموز

## 🚀 كيفية الاستخدام / How to Use

### 1. فتح التطبيق
افتح ملف `index.html` في أي متصفح ويب حديث.

### 2. تحميل أو التقاط صورة
- **للتحميل**: اضغط على "تحميل من الجهاز" أو اسحب الصورة إلى المنطقة المحددة
- **للالتقاط**: اضغط على "التقط صورة" واسمح للتطبيق بالوصول للكاميرا

### 3. تحليل المنتج
بعد اختيار الصورة، اضغط على "تحليل المنتج" لبدء عملية التحليل.

### 4. عرض النتائج
ستظهر النتائج متضمنة:
- التقييم الصحي للمنتج
- معلومات المنتج الأساسية
- القيم الغذائية التفصيلية
- قائمة المكونات
- التحذيرات والملاحظات

## 🛠️ التقنيات المستخدمة / Technologies Used

- **HTML5**: هيكل التطبيق
- **CSS3**: التصميم والتنسيق مع Grid و Flexbox
- **JavaScript (ES6+)**: المنطق والتفاعل
- **Font Awesome**: الرموز والأيقونات
- **Google Fonts**: خطوط Cairo و Amiri للعربية
- **Canvas API**: معالجة الصور
- **MediaDevices API**: الوصول للكاميرا

## 📱 المتطلبات / Requirements

- متصفح ويب حديث يدعم:
  - ES6+ JavaScript
  - Canvas API
  - MediaDevices API (للكاميرا)
  - CSS Grid و Flexbox
- اتصال بالإنترنت (لتحميل الخطوط والرموز)
